구글 스프레드시트 클라우드 연동

해당 웹앱은 서버를 직접 두지 않고, 구글 스프레드시트 매크로 (Google Apps Script)를 백엔드 노드로 이용합니다. 암호화된 데이터를 영구 보관하려면 아래 세팅을 진행해 주세요.

잠겨있음
💡 서버 환경변수에 연동 주소가 상시 고정되어 있습니다. 추가 입력 없이 상시 자동 프록시 동기화가 작동합니다.
🔒 마스터 설정 비밀번호(서버상 MASTER_PASSCODE)를 입력하여 동기화 세팅을 편집하거나 구글 스크립트 배포 주소를 변경하세요.
편집 권한이 승인되었습니다.

※ 주소를 넣고 저장하면 로컬 데이터가 구글에 전송되며, 이후 로그인 시 자동으로 구글 시트에서 최신 데이터를 연동 및 안전하게 복호수신합니다.

💡 3분 구글 시트 설치 가이드:

  1. 사용하려는 구글 스프레드시트 생성
  2. 상단 메뉴 확장 프로그램 > Apps Script 클릭
  3. 기존 코드 지우고 아래 제공된 스프레드시트 전용 매크로 스크립트 붙여넣기
  4. 상단 우측 배포 > 새 배포 선택
  5. 유형: 웹 앱, 액세스 가능한 사용자: 모든 사용자 (Any)로 설정 후 배포 클릭
  6. 발급된 웹 앱 URL 주소를 복사해서 위 SCRIPT_URL 란에 삽입 후 저장
