이 서비스는 어떠한 외부 연동 서버나 구글 데이터 추적 트래커로 작성 기록을 전송하지 않으며, 모든 마일스톤 및 수행기록은 본인 브라우저의 전용 안전 영역(Local Storage)에 100% 로컬 프라이버시가 존중되는 환경으로 보관됩니다. 기기를 교체하고 싶으시거나 데이터를 보존하시려면 우측의 [데이터 백업 및 다운로드] 메뉴를 적극 이용하세요.

📢

데이터 보관 및 복원 관리

💡 각 데이터베이스 항목을 분리된 CSV 파일 형태로 즉시 내려받습니다. 이 파일들은 개인 주소나 엑셀 시트로 자유롭게 업로드하여 커스텀 통계를 매기는 등 영구 귀속할 수 있습니다.

⚙️ 작성한 모든 정보(설정된 목표 리스트, 행동의 종류, 체크 잔디밭 누적 로그)를 규격화된 한 장의 통합 백업 JSON 파일로 원스톱 패키징합니다. 기기 이전용 보관을 완벽 보장합니다.

⚠️ 경고: 데이터를 복원하게 되면 현재 브라우저의 내부 가든 영역에 적힌 세부 사항들이 전부 삭제되고 업로드한 .json 백업본 내용으로 완전히 강제 교체됩니다. 이 과정은 실행 즉시 반영되어 복구할 수 없습니다.

🔒 100% 온디바이스 오프라인 보관 방식 안내

이 웹앱은 모든 설정 정보와 성취 이력(잔디밭 기록)을 어떠한 외부 클라우드나 외부 서버 없이 철저하게 각 브라우저 내부(LocalStorage)에만 안전하게 격리 보관합니다. 다른 사람과 데이터가 겹치지 않고 온전히 개인 전용 프로필 상태를 보장합니다.

🎁 실천 메커니즘을 친구와 공유하는 방법

  1. 'JSON 통합백업' 탭으로 이동하여 [백업용 통합 JSON 파일 받기]를 통해 다운로드 받거나 [클립보드에 통합 JSON 복사하기]를 클릭하세요.
  2. 복사되거나 저장된 JSON 파일을 카카오톡, 이메일 등으로 친구에게 공유하세요.
  3. 해당 자료를 전달받은 친구가 자신의 앱에서 '통합 데이터 복구' 탭을 통해 업로드 및 반영하면, 실천 목표와 세부 규칙들이 똑같이 세팅됩니다!
